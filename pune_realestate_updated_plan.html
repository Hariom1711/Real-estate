Every tool chosen for a reason — no unnecessary complexity.
Framework
Next.js 15
App Router, RSC, Server Actions, ISR caching — everything in one framework.
Database
Neon DB (Postgres)
Serverless Postgres. Free tier generous. Branches for dev/prod. No connection pooling headaches.
ORM
Prisma
Type-safe client, auto migrations, Prisma Studio as a visual DB editor — perfect for this project.
Auth
NextAuth v5
Credentials provider for admin login. Single hashed password in DB. Middleware protects /admin/*.
Images
Cloudinary
Free 25 GB. Signed upload from admin. Auto-optimize & resize via URL params. No storage cost on Vercel.
Styling
Tailwind CSS v4
Utility-first. Zero runtime CSS. Works perfectly with RSC. You already know it.
Validation
Zod + React Hook Form
Zod schemas shared between server actions and client forms. One source of truth for validation.
Hosting
Vercel
Zero-config Next.js deploy. Edge network. Free SSL. Connects to Neon natively.
Also: lucide-react for icons · next-themes for dark mode · @tanstack/react-table for admin leads table · resend for email notifications on new leads
